2013年10月15日

图片浏览软件 

开发环境：window 7，FLASH CS6

注意：
图片路径以及展示标题均从data.xml文件输入，图片放在images文件夹中。

//////////////////////////////////////////////////////////////////////////////
//// 功能：                                                                ////        
////（1）鼠标点击左右按钮，以及鼠标滚轮均可实现控制图片浏览；             ////
////（2）点击键盘左右方向键，也可实现控制图片浏览，并且会有另一种体验；   ////
////（3）简洁明了的自定义展示主题以及图片说明；                           //// 
////（4）图片资源无需预先调整大小，软件自动调整。                         ////
////（5）外部XML文件导入资源，随意修改展示内容；                          //// 
//////////////////////////////////////////////////////////////////////////////
